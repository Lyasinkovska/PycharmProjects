Preprocessing Open on Hyperskill

Preprocess an input text:

  • delete punctuation symbols (comma, period, exclamation and question marks),
  • remove unnecessary whitespaces both at the beginning and at the end,
  • convert all symbols to lowercase.

Then print your text.

[HINT]You can use the following tip to remove the punctuation marks: str.replace("!", ""). Replacing a symbol with an empty line is the same as removing it.[/HINT]


Sample Input:
Nobody expects the Spanish inquisition!
Sample Output:
nobody expects the spanish inquisition


Memory limit: 256 Mb
Time limit: 5s
